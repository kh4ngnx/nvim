# Requirement
- [Neovim](https://neovim.io) >= `0.9.4` - duh
- [Git](https://git-scm.com/install/)
- A [Nerd Font](https://nerdfonts.com) - ***JetBrainMono Nerd Font***
- Icon: [nvim-web-devicons](https://github.com/nvim-tree/nvim-web-devicons)
- A C compiler - [MSYS2](https://www.msys2.org/) or [w64devkit](https://github.com/skeeto/w64devkit)
- For `Snacks.nvim`:
    - [rg](https://github.com/BurntSushi/ripgrep#installation) for `grep()`
    - [fd](https://github.com/sharkdp/fd) for `picker()`
